# Async Patterns Lab ⚡

Academic research comparing async/concurrent patterns in PHP 8.2+: **Fibers**, **Amphp**, **ReactPHP**, and **Swoole**.

## Features
✅ PHP 8.2+ Fibers implementation  
✅ Amphp v3 async framework  
✅ ReactPHP event loop  
✅ Performance benchmarks  
✅ Concurrency patterns  
✅ Load testing examples  

## Installation
```bash
composer require syeedalireza/async-patterns-lab --dev
```
